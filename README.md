# Мой Блог - Django проект

Учебный проект на Django, включающий блог и книгу отзывов.

## Задания

### Блог (Практика 2)
- Главная страница со списком постов
- Страница "О блоге"
- Страница отдельного поста
- Наследование шаблонов от базового
- Статические файлы (CSS, изображения)

### Книга отзывов (Практика 3)
- Модель отзыва с полями: ФИО, email, текст отзыва, статус проверки
- Форма добавления нового отзыва
- Отображение только проверенных отзывов
- Панель администратора для модерации отзывов

## Установка и запуск

### Требования
- Python 3.10+

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone <url-репозитория>
cd django
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
```

3. Активируйте виртуальное окружение:
```bash
venv\Scripts\activate
```

4. Установите зависимости:
```bash
pip install -r requirements.txt
```

5. Примените миграции:
```bash
python manage.py migrate
```

6. Создайте суперпользователя (для доступа в админ-панель):
```bash
python manage.py createsuperuser
```

7. Запустите сервер разработки:
```bash
python manage.py runserver
```

8. Откройте в браузере:
- Главная страница: http://127.0.0.1:8000/
- О блоге: http://127.0.0.1:8000/about/
- Книга отзывов: http://127.0.0.1:8000/guestbook/
- Админ-панель: http://127.0.0.1:8000/admin/

## Структура проекта

`blog/urls.py` — маршруты для главной, «о блоге», поста и отзывов.
`templates/` — базовый шаблон и шаблоны страниц (наследование подключено).
`static/` — стили и изображения, подключаются через `{% templatetag openblock %} load static {% templatetag closeblock %}`.
`blog/models.py` — модель отзыва с полями ФИО, email, текст, флаг проверки.
`blog/forms.py` — форма добавления отзыва (валидация обязательных полей).
`blog/admin.py` — регистрация модели в админке.
`requirements.txt` — зафиксированные версии библиотек.

## Использование админ-панели

1. Перейдите по адресу http://127.0.0.1:8000/admin/
2. Войдите под учетными данными суперпользователя
3. В разделе "Отзывы" можно просматривать и модерировать отзывы
4. Установите галочку "Проверено" для отображения отзыва на сайте
